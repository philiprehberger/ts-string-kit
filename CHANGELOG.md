# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-03-20

### Added

- `truncate` — truncate strings with word-boundary support and position (end/middle/start)
- `template` — simple `{key}` placeholder interpolation
- `pluralize` — count-based singular/plural selection
- `wordWrap` — wrap text at word boundaries to a given width
- `pad` — pad strings left, right, or center
- `stripAnsi` — remove ANSI escape codes
- `stripHtml` — remove HTML tags
- `indent` — add leading whitespace to each line
- `dedent` — remove common leading whitespace
- `escapeHtml` — escape special HTML characters
- `unescapeHtml` — unescape HTML entities
- `countWords` — count words in a string
- `countLines` — count lines in a string
